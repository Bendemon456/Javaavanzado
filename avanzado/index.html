Javascript avanzado

JavaScript es un lenguaje de programación que se utiliza principalmente para crear páginas web interactivas. Gracias a JavaScript, los sitios web pueden reaccionar a las acciones del usuario, mostrar animaciones, validar formularios, cambiar contenido sin recargar la página y mucho más. Fue creado en 1995 y hoy es uno de los lenguajes más importantes del desarrollo web moderno. Funciona junto con HTML (estructura) y CSS (diseño), formando el trío básico de la web.
